# Federated Learning Based XSS Detection System

## Overview
This project implements a Federated Learning based XSS (Cross-Site Scripting) Detection System using Deep Learning.

The project uses:
- Flower Federated Learning Framework
- PyTorch
- CNN + BiLSTM + Attention Model

The system trains multiple distributed clients without sharing raw data.

---

## Features
- Federated Learning Simulation
- Deep Learning based XSS Detection
- Privacy Preserving Training
- Multiple Client Architecture
- Centralized Evaluation

---

## Technologies Used
- Python
- PyTorch
- Flower (flwr)
- NumPy
- Pandas
- Scikit-learn
